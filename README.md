Dans le dossier suivant se trouve tous les dossiers et fichiers nécéssaires aux fonctionnement du projet, dont le fichier dbProjet.sql, qui devra être exécuté au préalable sur la base de données de l'établissement.
Si vous exécutez le fichier sur une autre base de données, veuillez modifier le fichier connexion.php et remplacer le login, le mot de passe, le moteur de la BDD, le nom de la BDD et son adresse si besoin.
